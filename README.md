## 🚀 快速开始

### 环境要求
- Docker 20.10+
- Docker Compose 2.0+

### 部署步骤

```bash
# 1. 克隆项目
git clone https://github.com/Qioooba/family-app.git
cd family-app

# 2. 启动服务
cd docker
cp .env.example .env
docker-compose up -d --build

# 3. 查看状态
docker-compose ps
```

### 访问服务

| 服务 | 地址 |
|------|------|
| 前端页面 | http://localhost |
| 后端API | http://localhost:8080 |
| API文档 | http://localhost:8080/swagger-ui.html |
