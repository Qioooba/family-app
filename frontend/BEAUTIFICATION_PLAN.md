# 🎨 Family App 前端美化计划

## 目标风格
**润滑、丝滑、圆润、小清新、好看**

## 美化维度

### 1. 🟡 圆角优化
- 卡片圆角：20rpx → 24rpx（更圆润）
- 按钮圆角：保持 40rpx（全圆角）
- 输入框圆角：10rpx → 16rpx
- 标签圆角：8rpx → 12rpx（更柔和）
- 图标容器圆角：24rpx → 28rpx

### 2. 🌫️ 阴影改进
- 卡片阴影：更柔和的多层阴影
- 悬浮效果：hover时阴影加深
- 按钮阴影：添加微妙阴影增加层次感

### 3. ✨ 过渡动画
- 卡片hover：scale + shadow 过渡
- 按钮点击：scale 0.95 反馈
- 页面切换：淡入动画
- 列表项： stagger 进入动画

### 4. 🎨 配色优化（小清新风格）
- 主色调：#5B8FF9 → 更柔和的 #6B8DD6
- 背景色：#f5f6fa → 更温暖的 #f8f9fc
- 渐变色：使用更柔和的渐变
- 功能色：降低饱和度，更清新

### 5. 📐 间距优化
- 卡片间距：增加呼吸感
- 内边距：更舒适的阅读空间
- 元素间距：一致的8px基准

### 6. 🔤 字体优化
- 标题字重：600 → 500（更轻盈）
- 行高：1.6 → 1.5（更紧凑）
- 字体平滑：添加抗锯齿
