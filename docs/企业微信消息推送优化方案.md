# 企业微信消息推送优化方案

## 📅 讨论时间
2026-03-12

## 🎯 目标
在微信企业微信插件中，实现点击消息先展示详情，再选择是否跳转小程序。

---

## ✅ 方案确定

### 消息类型
**全部使用图文消息（news）**

### 流程
```
微信消息列表
    ↓ 点击
H5详情页（显示完整任务信息）
    ↓ 用户点击按钮
打开小程序 / 关闭
```

### 消息格式
```json
{
  "msgtype": "news",
  "news": {
    "articles": [{
      "title": "🏠 新任务：记得买牛奶",
      "description": "来自：陶陶\n时间：今天 18:00\n\n点击查看详情",
      "url": "https://qioba.cn:8443/notify-detail.html?type=task&id=123&token=xxx",
      "picurl": "https://qioba.cn:8443/logo.png"
    }]
  }
}
```

---

## 📋 任务类型统一处理

| 场景 | 消息标题 | 描述内容 | 按钮 |
|------|---------|---------|------|
| 创建任务 | 🏠 新任务：xxx | 来自：xxx<br>时间：xxx<br><br>点击查看详情 | 【去处理】→ 小程序 |
| 指派任务 | 🏠 新任务：xxx | 来自：xxx<br>被指派人：xxx<br><br>点击查看详情 | 【去处理】→ 小程序 |
| 完成任务 | ✨ 任务完成 | 执行人：xxx<br>任务：xxx<br><br>点击查看详情 | 【查看详情】→ 可选跳转 |
