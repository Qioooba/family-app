# 个人提醒中心系统设计方案

## 📅 设计时间
2026-03-12

## 🎯 目标
构建统一的个人提醒管理平台，支持多种提醒类型，每个人可独立设置自己的提醒。

---

## 📋 提醒类型分类

### 1. 生活提醒（日常类）
| 类型 | 说明 | 触发方式 | 频率 |
|------|------|---------|------|
| **喝水提醒** | 定时提醒喝水 | 固定时间 | 每天多次（8点、10点、14点等） |
| **久坐提醒** | 久坐后提醒活动 | 计时器 | 每小时 |
| **用眼提醒** | 用眼过度提醒休息 | 计时器 | 每45分钟 |

### 2. 物品管理（物品类）
| 类型 | 说明 | 触发方式 | 频率 |
|------|------|---------|------|
| **保质期提醒** | 食品/物品过期前提醒 | 到期前N天 | 一次性 |
| **药品提醒** | 用药时间提醒 | 固定时间 | 每天多次 |
| **物品更换提醒** | 牙刷、毛巾等更换周期 | 到期前N天 | 周期性（3个月、6个月） |

### 3. 财务提醒（财务类）
| 类型 | 说明 | 触发方式 | 频率 |
|------|------|---------|------|
| **贷款还款提醒** | 房贷/信用卡还款提醒 | 到期前N天 | 每月 |
| **账单提醒** | 水电煤缴费提醒 | 到期前N天 | 每月 |
| **理财到期提醒** | 理财产品到期提醒 | 到期前N天 | 一次性 |

### 4. 社交提醒（人际类）
| 类型 | 说明 | 触发方式 | 频率 |
|------|------|---------|------|
| **生日提醒** | 家人生日提醒 | 到期前N天 | 每年 |
| **纪念日提醒** | 结婚纪念日等 | 到期前N天 | 每年 |
| **节日提醒** | 春节、母亲节等 | 到期前N天 | 每年 |
| **拜访提醒** | 定期拜访长辈 | 固定时间 | 每月/每周 |

### 5. 智能家居（设备类）
| 类型 | 说明 | 触发方式 | 频率 |
|------|------|---------|------|
| **米家设备提醒** | 设备状态异常/耗材更换 | 事件触发 | 不定期 |
| **清洁提醒** | 扫地机清理尘盒 | 使用后N次 | 每N次 |
| **滤芯更换提醒** | 净水器/空气净化器滤芯 | 使用时间 | 每N个月 |

## ⚙️ 核心算法设计

### 1. 计算下次提醒时间
```java
public Date calculateNextRemindTime(ReminderConfig config) {
    CycleConfig cycle = config.getCycleConfig();
    Date now = new Date();
    
    switch (cycle.getCycleType()) {
        case "DAILY":
            // 每天固定时间
            return getTodayAtTime(config.getRemindTime());
            
        case "WEEKLY":
            // 每周指定星期几
            List<Integer> weekDays = cycle.getWeekDays();
            return getNextWeekDayAtTime(weekDays, config.getRemindTime());
            
        case "INTERVAL":
            // 每隔N天（用药）
            int interval = cycle.getIntervalDays();
            Date lastRemind = config.getLastRemindTime();
            return addDays(lastRemind, interval);
            
        case "ONCE":
            // 一次性提醒
            return config.getRemindDate();
            
        case "EXPIRY":
            // 到期前N天（保质期）
            Date expiryDate = config.getRelationData().getExpiryDate();
            return addDays(expiryDate, -config.getRemindDaysBefore());
    }
}
```

### 2. 扫描到期提醒任务
```java
@Scheduled(cron = "0 * * * * ?") // 每分钟执行
public void scanDueReminders() {
    Date now = new Date();
    Date fiveMinutesLater = addMinutes(now, 5);
    
    // 查询5分钟内要到期的提醒
    List<ReminderConfig> dueReminders = reminderMapper
        .selectDueReminders(now, fiveMinutesLater);
    
    for (ReminderConfig reminder : dueReminders) {
        // 异步执行提醒
        reminderExecutor.execute(() -> {
            sendReminder(reminder);
            updateNextRemindTime(reminder);
        });
    }
}
```

- 如：9点提醒喝水，点"稍后"→10点再次提醒
