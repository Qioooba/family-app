# 个人提醒中心系统设计方案

## 📅 设计时间
2026-03-12

## 🎯 目标
构建统一的个人提醒管理平台，支持多种提醒类型，每个人可独立设置自己的提醒。

---

## 📋 提醒类型分类

### 1. 生活提醒（日常类）
| 类型 | 说明 | 触发方式 | 频率 |
|------|------|---------|------|
| **喝水提醒** | 定时提醒喝水 | 固定时间 | 每天多次（8点、10点、14点等） |
| **久坐提醒** | 久坐后提醒活动 | 计时器 | 每小时 |
| **用眼提醒** | 用眼过度提醒休息 | 计时器 | 每45分钟 |

### 2. 物品管理（物品类）
| 类型 | 说明 | 触发方式 | 频率 |
|------|------|---------|------|
| **保质期提醒** | 食品/物品过期前提醒 | 到期前N天 | 一次性 |
| **药品提醒** | 用药时间提醒 | 固定时间 | 每天多次 |
| **物品更换提醒** | 牙刷、毛巾等更换周期 | 到期前N天 | 周期性（3个月、6个月） |

### 3. 财务提醒（财务类）
| 类型 | 说明 | 触发方式 | 频率 |
|------|------|---------|------|
| **贷款还款提醒** | 房贷/信用卡还款提醒 | 到期前N天 | 每月 |
| **账单提醒** | 水电煤缴费提醒 | 到期前N天 | 每月 |
| **理财到期提醒** | 理财产品到期提醒 | 到期前N天 | 一次性 |

### 4. 社交提醒（人际类）
| 类型 | 说明 | 触发方式 | 频率 |
|------|------|---------|------|
| **生日提醒** | 家人生日提醒 | 到期前N天 | 每年 |
| **纪念日提醒** | 结婚纪念日等 | 到期前N天 | 每年 |
| **节日提醒** | 春节、母亲节等 | 到期前N天 | 每年 |
| **拜访提醒** | 定期拜访长辈 | 固定时间 | 每月/每周 |

### 5. 智能家居（设备类）
| 类型 | 说明 | 触发方式 | 频率 |
|------|------|---------|------|
| **米家设备提醒** | 设备状态异常/耗材更换 | 事件触发 | 不定期 |
| **清洁提醒** | 扫地机清理尘盒 | 使用后N次 | 每N次 |
| **滤芯更换提醒** | 净水器/空气净化器滤芯 | 使用时间 | 每N个月 |

## 🗄️ 数据库表设计

### 表1：提醒配置主表（reminder_config）
```sql
CREATE TABLE reminder_config (
    id BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT '主键ID',
    user_id BIGINT NOT NULL COMMENT '用户ID',
    
    -- 提醒基础信息
    reminder_code VARCHAR(50) NOT NULL COMMENT '提醒编码：WATER_MORNING / MEDICINE_NOON / FOOD_EXPIRE_001',
    reminder_type VARCHAR(30) NOT NULL COMMENT '提醒类型：LIFE_DAILY / ITEM_MANAGE / FINANCE / SOCIAL / SMART_HOME',
    reminder_subtype VARCHAR(30) COMMENT '子类型：WATER / MEDICINE / FOOD_EXPIRE / LOAN / BIRTHDAY / MIJIA',
    
    title VARCHAR(100) NOT NULL COMMENT '提醒标题：如"早上喝水"',
    content VARCHAR(500) COMMENT '提醒内容：如"记得喝一杯温水，开启美好一天"',
    icon VARCHAR(50) COMMENT '图标：💧💊🎂🏠',
    
    -- 提醒时间配置
    remind_time VARCHAR(20) COMMENT '提醒时间：08:00（单次/每日）',
    remind_date DATE COMMENT '提醒日期（一次性提醒）',
    remind_days_before INT DEFAULT 0 COMMENT '提前几天提醒（到期类）',
    
    -- 周期配置（JSON格式，灵活扩展）
    cycle_config JSON COMMENT '周期配置：{
        "cycleType": "DAILY",      // DAILY-每天 WEEKLY-每周 MONTHLY-每月 YEARLY-每年 ONCE-一次性
        "weekDays": [1, 3, 5],      // 周几提醒（周一=1）
        "monthDays": [1, 15],       // 每月几号
        "intervalDays": 3           // 每隔几天（药品用）
    }',
    
    -- 关联数据（不同类型关联不同数据）
    relation_type VARCHAR(30) COMMENT '关联类型：TASK / FOOD / MEDICINE / DEVICE / ANNIVERSARY',
    relation_id BIGINT COMMENT '关联数据ID',
    relation_data JSON COMMENT '关联数据详情（冗余存储，避免多表查询）',
    
    -- 提醒方式
    push_type VARCHAR(20) DEFAULT 'WECHAT_WORK' COMMENT '推送方式：WECHAT_WORK / APP_PUSH / ALL',
    push_before_minutes INT DEFAULT 0 COMMENT '提前多少分钟推送',
    
    -- 状态控制
    status TINYINT DEFAULT 1 COMMENT '状态：0-停用 1-启用 2-暂停',
    priority TINYINT DEFAULT 5 COMMENT '优先级：1-10，数字越大优先级越高',
    
    -- 执行记录
    last_remind_time DATETIME COMMENT '上次提醒时间',
    last_remind_status VARCHAR(20) COMMENT '上次提醒状态：SUCCESS / FAILED / SKIPPED',
    next_remind_time DATETIME COMMENT '下次提醒时间',
    remind_count INT DEFAULT 0 COMMENT '已提醒次数',
    max_remind_count INT DEFAULT 0 COMMENT '最大提醒次数（0为无限）',
    
    -- 创建信息
    create_by BIGINT COMMENT '创建人ID（可为他人创建）',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
    update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    
    UNIQUE KEY uk_user_code (user_id, reminder_code),
    INDEX idx_user_type (user_id, reminder_type),
    INDEX idx_next_time (next_remind_time),
    INDEX idx_status (status)
) COMMENT='个人提醒配置表';
```

## ⚙️ 核心算法设计

### 1. 计算下次提醒时间
```java
public Date calculateNextRemindTime(ReminderConfig config) {
    CycleConfig cycle = config.getCycleConfig();
    Date now = new Date();
    
    switch (cycle.getCycleType()) {
        case "DAILY":
            // 每天固定时间
            return getTodayAtTime(config.getRemindTime());
            
        case "WEEKLY":
            // 每周指定星期几
            List<Integer> weekDays = cycle.getWeekDays();
            return getNextWeekDayAtTime(weekDays, config.getRemindTime());
            
        case "INTERVAL":
            // 每隔N天（用药）
            int interval = cycle.getIntervalDays();
            Date lastRemind = config.getLastRemindTime();
            return addDays(lastRemind, interval);
            
        case "ONCE":
            // 一次性提醒
            return config.getRemindDate();
            
        case "EXPIRY":
            // 到期前N天（保质期）
            Date expiryDate = config.getRelationData().getExpiryDate();
            return addDays(expiryDate, -config.getRemindDaysBefore());
    }
}
```

### 2. 扫描到期提醒任务
```java
@Scheduled(cron = "0 * * * * ?") // 每分钟执行
public void scanDueReminders() {
    Date now = new Date();
    Date fiveMinutesLater = addMinutes(now, 5);
    
    // 查询5分钟内要到期的提醒
    List<ReminderConfig> dueReminders = reminderMapper
        .selectDueReminders(now, fiveMinutesLater);
    
    for (ReminderConfig reminder : dueReminders) {
        // 异步执行提醒
        reminderExecutor.execute(() -> {
            sendReminder(reminder);
            updateNextRemindTime(reminder);
        });
    }
}
```

- 冰箱食品过期提醒 → 共享给全家人
- 房贷还款提醒 → 夫妻双方都收到

- 尤其适合用药提醒、重要事项

- 点击"稍后提醒"，智能选择延后时间
- 如：9点提醒喝水，点"稍后"→10点再次提醒
