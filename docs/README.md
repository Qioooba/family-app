# 文档目录

为保持项目根目录简洁，辅助文档统一收敛到 `docs/` 下。

## 目录说明

- `guides/`
  - 面向使用和调试的操作文档
  - 包括配置说明、快速开始、调试指南
- `reference/`
  - 面向项目理解的参考文档
  - 包括项目说明、功能清单
- `releases/`
  - 面向版本演进的更新记录
- 根目录 `docs/*.md`
  - 保留专题设计、接入说明、业务方案等长期参考文档
